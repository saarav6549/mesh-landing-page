# MESH - פלטפורמת גילוי אופנה מבוססת AI

פלטפורמת גילוי אופנה מבוססת AI שמחברת בין השראת סטייל לקנייה חכמה.

## תכונות

- 🔍 חיפוש חזותי AI
- 👥 תוכן משפיענים
- ❤️ ארון וירטואלי
- ✨ פיד מותאם אישית
- 🛍️ קניה ישירה
- 👥 קהילת אופנה

## טכנולוגיות

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## התקנה

```bash
npm install
```

## הרצה

```bash
npm run dev
```

## בנייה

```bash
npm run build
```
